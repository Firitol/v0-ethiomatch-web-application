## 🎯 Quick Reference

### Setup
- `npm install` - Install dependencies
- `npm run dev` - Run dev server
- Browser: `http://localhost:3000`

### Key Files
- Discover page: `app/page.tsx`
- Messaging page: `app/matches/page.tsx`
- Profile page: `app/profile/page.tsx`
- Database: `lib/db.ts`
- Auth: `app/auth-context.tsx`
- Swipe card: `components/swipe-card.tsx`

### Common Edits
- Change colors: Edit `app/globals.css`
- Modify features: Edit relevant `app/*/page.tsx`
- Add new page: Create `app/[route]/page.tsx`
- Update database: Edit `lib/db.ts`
